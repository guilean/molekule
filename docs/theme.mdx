# Theming

Refractal comes with a default theme. You can override any of these configurations by passing a `theme` object to `<ThemeProvider />`

```
{
  radii: [0, 2, 4],
  typography: {
    fontSize: 14,
  },
  colors: {
    primary: '#5759D6',
    green: '#1dd1a1',
    red: '#ee5253',
    blue: '#48dbfb',
    orange: '#feca57',
    grayLight: '#e9eef4',
    grayMid: '#8395a7',
    grayDark: '#576574',
  },
  variants: {
    success: 'green',
    danger: 'red',
    info: 'blue',
    warning: 'orange',
  },
  heights: {
    xs: 20,
    sm: 28,
    md: 32,
    lg: 40,
    xl: 44,
  },
  fontSizes: {
    xs: 10,
    sm: 10,
    md: 12,
    lg: 14,
    xl: 16,
  },
  breakpoints: [386, 768, 1024],
};
```
